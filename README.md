# Proyecto final de la Lic. en Sistemas de la Universidad del Salvador

## Sistema de Gestión de Órdenes de Trabajo - SGOT

Sistema desarrollado para un laboratorio de electrónica (reparación de equipos, calibración, etc), cuyo principal objetivo es optimizar los tiempos de entrega.
Permite gestionar la recepción de equipos, centralizando la información y mejorando el control operativo.

## Tecnologías y conceptos utilizados

* Lenguaje: [Java / UML]
* Programación Orientada a Objetos (POO)
* Arquitectura en capas (presentación / negocio / datos)
* Base de datos relacional (MySQL)
* Diagramas UML (casos de uso / clases)
* Manejo de errores y validaciones


## Mi rol en el proyecto

* Análisis de requerimientos
* Diseño del modelo de datos y clases
* Desarrollo de funcionalidades principales
* Implementación de validaciones y reglas de negocio
* Pruebas funcionales del sistema

## Decisiones técnicas y aprendizajes
###### Durante el desarrollo del proyecto:
* Se definió una arquitectura en capas para mejorar la mantenibilidad
* Se implementaron validaciones para garantizar la consistencia de los datos
* Se identificaron escenarios de error y se definieron mensajes claros para el usuario
* Se adquirió experiencia en el diseño de sistemas similares a los utilizados en entornos reales de trabajo
 
## Casos de uso / escenarios relevantes

* Creación y gestión de Usuario, Cliente, Equipo, Orden de Trabajo
* Modificación de estados según reglas del sistema
* Manejo de errores ante datos inválidos
* Consulta de información según permisos del usuario
